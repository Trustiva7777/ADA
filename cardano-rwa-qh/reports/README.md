# Reports

Use this folder to store allocation ledgers and payout proofs for investor audit.

Suggested structure
- allocations-YYYYQX.csv — wallet, units, tx hash
- quarterly-YYYYQX/
  - payout_plan.csv
  - proofs/
    - usdc_receipt_*.pdf or txids.json
  - statement.md
- monthly-YYYY-MM.md — ops summary
